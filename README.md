# uMarioMod

converting visual c++ to c++.

godMode and unlimited lives are added. 

## Build Pre-requisites

Ubuntu/Debian:

    $ apt install cmake libsdl2-dev libsdl2-image-2.0-0-dev libsdl2-mixer-2.0-0

## Building and running

    $ make build run

    # or

    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
    $ ./uMario
